Mode Commands
CMD-MODE-01
AUTO Mode
System manages source automatically. AUTO always takes priority and can override MANUAL.
CMD-MODE-02
MANUAL Mode
Operator selects source manually. Commands are still validated by AUTO before execution.
Source Commands (validated by ESP32)
CMD-SRC-01
Battery
Switch to battery/inverter. ESP32 validates: Vbat must be ≥ 10.5V and MANUAL mode active.
CMD-SRC-02
Grid (PLN)
Switch to grid power. ESP32 validates: grid must be available ≥ 180V and MANUAL mode active.
CMD-SRC-03
Load OFF
Disconnect all relays. Always permitted by AUTO. To reconnect, send Grid or Battery command.
